# Telegram Video Downloader Bot 🤖🎬

Этот бот позволяет скачивать видео с YouTube и других платформ по названию или ссылке. 
Если видео весит более 50 МБ, бот автоматически нарезает его на части по 7 минут для обхода лимитов Telegram.

## Основные функции:
* **Поиск и скачивание:** Просто отправьте название видео.
* **Авто-нарезка:** Разбивка тяжелых файлов на части (FFmpeg).
* **Чистка:** Бот удаляет временные файлы с сервера после отправки.

## Технологии:
* **Язык:** Python 3.10
* **Библиотеки:** `pyTelegramBotAPI`, `yt-dlp`
* **Инструменты:** `FFmpeg`, `Docker`

## Как запустить (Deploy):
1. Создайте бота в [@BotFather](https://t.me/BotFather) и получите токен.
2. Вставьте токен в `main.py`.
3. Залейте файлы на GitHub.
4. Подключите репозиторий к **Render.com** или **Koyeb**, выбрав среду **Docker**.
